# Enterprise RAG Project

This directory contains the complete Enterprise RAG system with Granite-3.1-8B-Instruct.

## Structure

- `data/` - Datasets and RAG documents
- `triton_kernels/` - Custom Triton FlashAttention implementation
- `scripts/` - Training, evaluation, and utility scripts
- `inference/` - Deployment and inference code
- `docker/` - Docker and Kubernetes configurations
